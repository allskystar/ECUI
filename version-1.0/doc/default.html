ECUI 开发手册

ECUI 的全称是 Enterprise Class UI，是一个用于页面交互控制的框架，类似于MFC/VCL等开发框架，对浏览器默认的事件与标签状态进行统一的管理。ECUI 的核心由公共操作库(提供事件管理、焦点控件、拖拽、缩放等功能)、基础控件两大部分组成，标准控件库是对基础控件行为的扩展与封装。
