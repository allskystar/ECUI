装饰器

装饰器，附着在控件外，为控件增加额外的展示区域，充分提升控件的展示效果。

使用方法

框架默认初始化方式

  1. 为控件设置相应的装饰器属性

    控件的decorate属性控制控件的装饰器――decorate:装饰器(装饰器基本样式)，如decorate:magic-decorator(caldtr) 表示为控件添加基本样式为caldtr的九宫格装饰器

  2. 设置相应的CSS样式

    每种装饰器的附加展示区域（HTML标签）都会有相对应的class属性，其命名方式是 装饰器基本样式 + 装饰器区域附加样式。具体实例请参见各装饰器详细介绍

代码手动添加方式

<body>
  <div eui="type:control;id:ctrl1"></div>
  <div class="lr" eui="type:control;id:ctrl2"></div>
  <div eui="type:control;id:ctrl3"></div>
  <div eui="type:control;id:ctrl4"></div>
</body>
<script>
  var dec = new ecui.ext.Decorator(ecui.get("ctrl1"), "dec");             //为ctrl1控件添加基本装饰器,装饰器基本样式为dec
  var lrdec = new ecui.ext.LRDecorator(ecui.get("ctrl2"));                 //为ctrl2控件添加左右装饰器，未特殊指定装饰器的基本样式，框架自动生成基本样式为lr-decorator
  var tbdec = new ecui.ext.TBDecorator(baid.ec.get("ctrl3"), "tbdec"));       //为ctrl3控件添加上下装饰器，装饰器基本样式为tbdec
  var mgcdec = new ecui.ext.MagicDecorator(baid.ec.get("ctrl4"), "mgcdec"));  //为ctrl4控件九宫格装饰器，装饰器基本样式为mgcdec
</script>
